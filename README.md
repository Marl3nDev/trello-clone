# Trello Clone

[![Build Status](https://github.com/Marl3nDev/trello-clone/actions/workflows/deploy.yml/badge.svg)](https://github.com/Marl3nDev/trello-clone/actions/workflows/deploy.yml)

Trello-подобное приложение для управления задачами с использованием Drag and Drop API.

## Демо

[Посмотреть демо](https://Marl3nDev.github.io/trello-clone/)

## Возможности

- Добавление, удаление и перемещение карточек
- Перетаскивание карточек между колонками
- Автосохранение в LocalStorage
- Адаптивный дизайн

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Marl3nDev/trello-clone.git
cd trello-clone
```

2. Установите зависимости:
```bash
yarn install
```
